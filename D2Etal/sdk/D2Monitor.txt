Codes and added features to D2MonitorBasic useSendCopyData(d2monitor, null, xxx, text);"d2monitor" is defined in D2Etal\scripts\Common\Libs\CLColorConverter.ntl"null" retain/use for each message as this defaults to the clients process id during sendxxx - codes available for specific useage0 - for any text message to be displayed - color encoding supported d2nt/etal compatibility333 - reserved for alerting/displaying SOJ sales666 - reserved for Dclone walks alert - loop or single play alert sound700 - reserved for when hostiled - loop or single play alert sound1000 - play cow audio wave - for custom alerting use3000 - play system beep - for custom audio alerting useEntry points have code added to change window title to char name when d2monitor is running.
